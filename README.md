# 七鱼访客端微信sdk工程
七鱼访客端sdk是基于taro的微信小程序插件项目。主要用于提供客户侧的微信小程序申请客服服务。对接文档[微信小程序sdk文档](http://qi.163.com/docs/guide/wechat_sdk/)

## 环境安装

1. 安装node.js

2. 安装@tarojs/cli
```bash
npm/cnpm install -g @tarojs/cli@1.3.14
```

经过taro的迭代，现在安装最新版@tarojs/cli可能项目编译会有问题，正在解决ing。。。

## 构建项目

1. 安装依赖包

```bash
npm/cnpm install
```
2. 开发模式下预览项目

```bash
npm run dev
```

3. 生产环境编译项目

```bash
npm run build 
```
